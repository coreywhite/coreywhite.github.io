Hello World

2016-01-31
Introduction

Hello, and welcome to my blog. My name is Corey White.

I have been coding on and off for over fifteen years. I have never been employed as a full time software engineer or programmer, but I have used programming as a tool throughout my career. I don’t have any major open source projects under my belt, and I am not an expert in any language. I know a little bit about a lot of things, but I don’t know a lot about any one programming language or ecosystem.

Why am I telling you this? To give this blog some context.

I am a “beginner with some background.” As I learn full stack web development in JavaScript, primarily through Free Code Camp, I am starting this blog to help myself and my fellow learners. I hope that by drawing on some of what I’ve learned in my coursework, my career, and my explorations of other programming languages, I will have enough background to bring some insight to the learning process.

So what’s this blog?

I spoiled this a bit above! At least for now, this blog will be about the process of learning full stack web development, especially through the Free Code Camp curriculum. Along the way, I expect to share some of the challenges I faced and the resources I used.

Why Free Code Camp?

I started working on Free Code Camp after listening to the founder, Quincy Larson, talk about the motivation for the program on an episode of the CodeNewbie podcast. Quincy’s description of his struggles while learning to code spoke to me (there’s a great long-form version of the story on his Medium). I agree with the Free Code Camp philosophy that the key to learning to code is to focus. It was a conclusion I had finally come to myself, just a few days before hearing the podcast. I needed to stop jumping between languages and projects and MOOCs and editors, always in search of something better, some “right thing to learn.” I just needed focus. And that’s what Free Code Camp has that other online resources for learning to code don’t.

There are other reasons too. I’m interested in full stack web development, and Free Code Camp definitely has a commitment to integrating real-world technologies. I really like the balance between guided lessons, short challenges, and long self-directed projects. I like the community aspects and the focus on networking and engaging with the wider world of development.

But mostly, it’s about the focus.

What to expect

  • Guidance on helpful things to learn that are a bit outside the Free Code Camp curriculum
    • But always with an eye on not losing focus!
  • Suggestions and links to learning resources I found helpful
  • Post-mortems of some of my Free Code Camp projects
    • What went wrong
    • What went right
    • What I learned
  • My thoughts on Free Code Camp and on learning to code
